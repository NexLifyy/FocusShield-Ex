Frequently Asked Questions

How does FocusShield block websites? +
FocusShield uses Chrome's secure declarativeNetRequest API to intercept and redirect requests to blocked domains. When you visit a blocked site, you're taken to a clean, local motivational screen, meaning no data ever leaves your browser.
Is FocusShield really free? +
Yes! The core features (platform blocking, custom URLs, and basic schedules) are completely free with no account needed. The Pro plan adds cloud sync, unlimited schedules, and advanced analytics.
Can I bypass the block screen? +
You can toggle blocking on and off from the extension popup at any time. For stronger commitment mode, Pro users can enable a password lock so blocks can't be bypassed without it.
Does it slow down my browser? +
No. FocusShield uses Chrome's native blocking API which operates at the network level instead of via injected JavaScript. This makes it extremely lightweight with zero measurable performance impact.
What happens to my data? +
Free plan data is stored locally in your browser. The Pro plan syncs settings to the cloud so you can use your configuration across multiple computers. We never sell or share your data. See our Privacy Policy for full details.
